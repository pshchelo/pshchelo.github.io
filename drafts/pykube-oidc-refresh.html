Why

Setting up OpenID IdP

I am starting completely fresh with Google’s API, so doing all from scratch (I do have google mail address though already).

  1. Go to Google Cloud console htps://console.developers.google.com
  2. Create new project (current quota seems to be 12 projects) k8s-test-project name, ID is that with random nums appended

Create new credentials

  • OAuth Client ID
  • Web Application
  • Authorized redirect URIs: http://127.0.0.1:64464/callback

Installing and configuring microk8s

https://microk8s.io/docs/configuring-services

api-server flags:

--oidc-issuer-url=https://accounts.google.com
--oidc-username-claim=email
--oidc-client-id=<client id from previous step>
